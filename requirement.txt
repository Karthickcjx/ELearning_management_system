Learning Management System - Project Requirements and Dependencies

This repository is not a Python project. This file is a single-place inventory
of the direct dependencies and runtime requirements declared in the project.

Source files checked:
- backend/pom.xml
- frontend/package.json
- backend/src/main/resources/application.yml
- README.md

1. Runtime requirements
- Java 17+
- Maven 3.6+ (or use backend/mvnw)
- Node.js + npm
- MySQL 8.0+

8. Notes
- Most entries are direct dependencies declared in package.json and pom.xml.
- autoprefixer is also listed because it is referenced by frontend/postcss.config.js
  and resolved in frontend/package-lock.json.
- Transitive npm and Maven dependencies are resolved automatically through
  package-lock.json and Maven dependency management.
