Advanced Features

Advanced Capabilities

🤖
AI-Powered Features
Google Gemini API integration for career roadmap generation, learning path recommendations, and AI-assisted content with retry mechanisms for high availability
🏠
Collaborative Rooms
Password-protected virtual rooms with real-time WebSocket communication (STOMP). Students create/join rooms using a room ID and password
🗺️
Career Roadmap
AI-generated structured career roadmaps with step-by-step learning paths, saving plans to database for future reference
🛡️
Admin Dashboard
Full admin panel: user management, course CRUD, announcements, content moderation, analytics, messages, and platform settings
📋
Assignments
Assignment submission and tracking system integrated with course management
⭐
Feedback System
Course feedback and rating mechanism allowing students to review and rate their learning experience

Security

Security Architecture

🔐 Authentication
JWT Token-based authentication
jjwt library version 0.12.6
Stateless session management
Token stored in browser (localStorage/cookie)
Default admin: admin@gmail.com / admin123
BCrypt password hashing
